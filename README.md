# + Closer

## アプリケーション概要

近年性別問わず家事参加への関心が高まる一方、初心者向けの家事分担アプリが存在しないことから、
家事初心者に向けたアプリを作成し家庭生活への積極的参加を支援する

# 目指した課題解決

このアプリケーションを利用することで、
「家事に関わりたいけど、関わり方がわからない」という家事初心者の人たちが
積極的に家事参加出来るようになるために、家事のやり方・家事の継続力を身につけることができる。
